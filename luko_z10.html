1. Wymień oraz opisz dwie części selektora

    Każda reguła CSS składa się z dwóch głównych części:

    • Selektor – określa, do jakich elementów HTML zostanie zastosowany styl, np. p, .klasa, #id.
    • Deklaracja – zawiera zestaw właściwości i wartości określających wygląd elementu.

  2. Znaczenie nawiasów klamrowych {...}

    Nawiasy klamrowe grupują deklaracje stylów dla danego selektora.

    p {
    color: red;
    font-size: 16px;
}

    Wszystkie właściwości i wartości umieszczone w klamrach będą miały wpływ na dany selektor.

  3. Opisz cechy (właściwości CSS)

    Cechy (właściwości) CSS określają wygląd elementów HTML.

    • color – zmienia kolor tekstu.
    • font-size – określa rozmiar czcionki.
    • margin – ustawia marginesy wokół elementu.
    • background-color – zmienia kolor tła.

  4. Opisz wartości

    Wartości definiują sposób działania cechy.

    • Dla color: red, #ff0000, rgb(255, 0, 0).
    • Dla font-size: 16px, 1.2em, 120%.
    • Dla margin: 10px, auto, 5% 10%.
  5. Kiedy stosujemy znak średnika (;), a kiedy nie jest konieczny?
    • Średnik oddziela poszczególne deklaracje w bloku stylu.
    • Jest wymagany między właściwościami w stylach CSS.
    • W ostatniej deklaracji w bloku może być pominięty, ale zaleca się jego stosowanie dla przejrzystości kodu.
