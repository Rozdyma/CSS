1) Gdzie jest wstawiany?

Wewnętrzny arkusz stylów CSS jest umieszczany w sekcji <head> dokumentu HTML, wewnątrz znacznika <style>.

2) Składnia teoretyczna

    <style>
        selektor {
            cecha: wartość;
        }
    </style>

3) Jakie mogą być selektory?

Selektory w CSS mogą być różne, np.:

  • Selektor elementu (np. p, h1, div)
  • Selektor klasy (np. .nazwa-klasy)
  • Selektor identyfikatora (np. #id-elementu)
  • Selektor atrybutu (np. [type="text"])
  • Selektor potomka, dziecka, sąsiedni i inne

4) Jakie mogą być cechy i wartości?

Cechy (właściwości) CSS mogą obejmować:

  • color: red;
  • font-size: 16px;
  • margin: 10px;
  • padding: 5px;
  • background-color: #f0f0f0;
