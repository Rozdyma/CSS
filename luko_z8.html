1. Gdzie jest wstawiany wewnętrzny arkusz stylów?

    2. Wewnętrzny arkusz stylów jest wstawiany w sekcji <head> dokumentu HTML wewnątrz znacznika <style>. Pozwala na definiowanie styli dla wielu elementów na jednej stronie.

  2. Składnia teoretyczna

    3. <head>
    <style>
        selektor {
            właściwość: wartość;
        }
    </style>
</head>

    Wewnętrzny arkusz stylów definiuje reguły CSS w obrębie dokumentu HTML, co pozwala na modyfikację wyglądu bez potrzeby używania zewnętrznych plików.

  3. Jakie mogą być selektory?

    4. W wewnętrznym arkuszu stylów można stosować różne selektory:

    • Selektor elementu – np. p, h1.
    • Selektor klasy – np. .nazwa-klasy.
    • Selektor ID – np. #nazwa-id.
    • Selektor atrybutów – np. input[type="text"].

  4. Jakie mogą być cechy i wartości?

    • Cechy (właściwości CSS) – definiują wygląd elementu, np. color, font-size, margin.
    • Wartości – określają sposób formatowania:
      • Dla color: red, #ff0000, rgb(255, 0, 0).
      • Dla font-size: 16px, 1.2em, 120%.
      • Dla margin: 10px, auto, 5% 10%.
