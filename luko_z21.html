Identyfikatory — ID

Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID to identyfikator – unikalna nazwa przypisana jednemu elementowi na stronie HTML.
Jaki jest cel stosowania ID? Umożliwia indywidualne stylizowanie i odwoływanie się do konkretnego elementu na stronie.
Jakie jest znaczenie znaku # Znak # w CSS oznacza, że stylujemy element po jego identyfikatorze (np. #id_nazwa).
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W HTML przypisujemy atrybut id="nazwa", a w CSS odnosimy się do niego poprzez #nazwa. Styl zostanie zastosowany tylko do tego jednego elementu.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. Elementy z ID muszą być unikalne – tylko jeden element może mieć dane ID.
  2. ID nie powinno być używane wielokrotnie na tej samej stronie.
  3. Jeśli element może pojawić się więcej niż raz, lepiej użyć klasy zamiast ID.
