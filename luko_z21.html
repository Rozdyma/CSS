Użycie ID w CSS

Pojęcie Co znaczy skrót ID? Jaki jest cel stosowania ID? Jakie jest znaczenie znaku #? Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Uwagi na temat stosowania ID (trzy konieczne wyliczenia).
Identyfikatory — ID Identyfikator Umożliwia unikalne formatowanie elementów na stronie. Oznacza, że definiujemy identyfikator w CSS. Wywołanie ID następuje poprzez użycie selektora z #, np. #nazwaID. 1. Identyfikator może wystąpić tylko raz w całym dokumencie.
2. Elementy oznaczone ID muszą być unikatowe.
3. W przypadku wątpliwości lepiej użyć klasy, która może być powielana.
