Definicja CSS - Kaskadowych Arkuszy Stylów

CSS - Kaskadowe Arkusze Stylów (z ang. Cascading Style Sheets) jest to specjalny język stworzony w celu elastycznego zarządzania sposobem formatowania (wyglądem) elementów znajdujących się na stronach WWW.

Zadanie 3: Cechy modyfikowania formatowania

HTML CSS
Modyfikacja wyglądu jest żmudna i chaotyczna. Rozdzielenie struktury i wyglądu ułatwia zarządzanie.
Formatowanie w kodzie HTML. Formatowanie w osobnym pliku CSS.

Nowe możliwości CSS:

  1. Lepsze zarządzanie wyglądem stron.
  2. Przejrzystszy kod.
  3. Łatwe zmiany dla wielu stron jednocześnie.

Wady CSS:

  • Złożona składnia nietolerująca błędów.
  • Brak sygnalizacji błędów.
  • Różna obsługa przez przeglądarki.
