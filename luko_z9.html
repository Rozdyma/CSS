1. Gdzie jest wstawiany zewnętrzny arkusz stylów?

    Zewnętrzny arkusz stylów jest wstawiany w sekcji <head> dokumentu HTML za pomocą znacznika <link>. Może być używany przez wiele stron jednocześnie.

  2. Składnia teoretyczna

    <head>
    <link rel="stylesheet" href="style.css">
</head>

    Plik CSS zawiera reguły stylów, które mogą być stosowane do różnych dokumentów HTML.

  3. Jak jest najprostsza forma wstawiania stylu zewnętrznego?

    <link rel="stylesheet" href="style.css">

    W ten sposób dołączamy plik CSS do dokumentu HTML.

  4. Ile arkuszy styli można dołączyć?

    Można dołączyć wiele arkuszy, ale obowiązuje zasada, że style załadowane później nadpisują wcześniejsze reguły.

  5. Co się dzieje w przypadku konfliktów?

    Jeżeli różne arkusze stylów definiują te same właściwości, priorytet ma ten, który został załadowany jako ostatni.

  6. Jak są tworzone i edytowane arkusze?
    • W edytorze tekstowym – np. Notepad++, Visual Studio Code.
    • W specjalnym edytorze CSS – np. Adobe Dreamweaver.
  7. Jakie rozszerzenie posiada plik arkusza styli?

    Plik CSS posiada rozszerzenie .css.

  8. Jakie są zalety zewnętrznego arkusza styli?
    • Umożliwia oddzielenie struktury HTML od prezentacji.
    • Jeden plik CSS może być stosowany do wielu stron.
  9. Uwagi na temat pisania w CSS

    Należy dbać o czytelność kodu, stosować komentarze i unikać nadmiarowego powielania reguł.
