Opis stylu zewnętrznego

1. Składnia teoretyczna

selektor {
    cecha: wartość;
}

Przykład:

p {
    color: blue;
    font-size: 16px;
}

2. Gdzie wstawiany jest styl zewnętrzny?

Styl zewnętrzny jest wstawiany w sekcji <head> dokumentu HTML przy użyciu znacznika <link>. Może on występować wielokrotnie, ale zazwyczaj używa się jednego głównego pliku CSS.

3. Najprostsza forma wstawiania stylu zewnętrznego

<link rel="stylesheet" href="style.css">

4. Ile arkuszy styli może być dołączonych?

  • Do jednego dokumentu HTML można dołączyć wiele arkuszy stylów.
  • Należy unikać konfliktów między stylami.
  • Style z plików dołączonych później mogą nadpisywać wcześniejsze.

5. Co się dzieje w przypadku konfliktów w stylach?

  1. Przeglądarka stosuje styl z arkusza, który został załadowany później.
  2. Można użyć reguły !important, aby wymusić zastosowanie konkretnego stylu.
  3. Styl zapisany w znaczniku style w dokumencie HTML ma wyższy priorytet niż arkusz zewnętrzny.

6. Jak są tworzone i edytowane pliki arkuszy za pomocą edytora tekstowego?

  • Pliki CSS można tworzyć w dowolnym edytorze tekstowym (np. Notatnik, VS Code, Sublime Text).
  • Plik powinien być zapisany z rozszerzeniem .css.
  • W celu podglądu zmian, plik CSS musi być dołączony do dokumentu HTML.
